# Classification-conditional sandwich covariance for the gate coefficients. V = Ainv B Ainv, with A the negative penalised-Q Hessian and B the score crossproduct. Independent-score (not spatial-dependence robust) – disclosed; the bootstrap is the default for spatially-dependent inference.

Classification-conditional sandwich covariance for the gate
coefficients. V = Ainv B Ainv, with A the negative penalised-Q Hessian
and B the score crossproduct. Independent-score (not spatial-dependence
robust) – disclosed; the bootstrap is the default for
spatially-dependent inference.

## Usage

``` r
gate_sandwich_vcov(Z, P, fit)
```
